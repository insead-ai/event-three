Code with Claude, Debriefed - INSEAD AI Club (Event 3)

HOW TO OPEN
  Double-click deck_master.html (opens in your web browser).

VIEWS
  - Normal: scroll through all the slides.
  - Presenter (one slide at a time): add ?slide=1 to the address bar,
    e.g.  .../deck_master.html?slide=1   (change the number to move)
  - Speaker notes: add ?notes to the address bar.

NOTE
  - Needs an internet connection for the fonts and for the two live
    "results" slides near the end (they embed a live poll dashboard).
  - The QR codes link to the live audience-poll pages and a few articles.
